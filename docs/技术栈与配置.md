```python
# apps/backend/config.py

# 类型选择: 'wearm' (串口) / 'hiwonder' (WiFi)
ARM_TYPE = 'wearm'

# WeArm (CH340 串口)
SERIAL_BAUD = 115200
SERIAL_PORT = '自动检测'   # CH340 VID:0x1A86 PID:0x7523

# Hiwonder ArmPi (WiFi)
HIWONDER_HOST = '192.168.1.175'
HIWONDER_PORT = 9999
```

### 逆运动学参数 (WeArm)

```python
ARM_H0 = 20.0    # 底座旋转轴到肩关节垂直距离 (mm)
ARM_L1 = 103.0   # 大臂长度 (mm)
ARM_L2 = 96.0    # 小臂长度 (mm)
ARM_L3 = 50.0    # 腕到末端执行器距离 (mm)
```

### 摄像头

# 自动检测策略: 遍历 index 0-4，分别测试 MSMF / DSHOW 后端

## 文档处理核心流程

```
拍照 → 纸张检测 → 二维码扫描 → OCR 字段提取
→ 自动分类 → 模板字段提取 → 多页检测 → 规则验证
→ 盖章定位(IK/标定/模板/默认) → 盖章 → 拍照存档 → 审计日志
```
