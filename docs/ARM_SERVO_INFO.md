# WeArm 机械臂舵机控制说明

## 硬件参数

| 参数 | 值 |
|------|-----|
| 型号 | WeArm 老款 |
| 通信 | PWM 文本协议（CH340 USB转串口） |
| 波特率 | 115200 |
| PWM 范围 | 500 ~ 2500 |
| 中位值 | 1500（ neutral_value） |

## 舵机定义

| ID | 名称 | 轴向 | 说明 |
|----|------|------|------|
| S0 | 底盘 | 垂直旋转 | 控制整体左右旋转，约270度范围 |
| S1 | 大臂 | 俯仰 | 决定机械臂前后俯仰角度，1500时垂直向上 |
| S2 | 小臂 | 俯仰 | 辅助大臂控制俯仰角度 |
| S3 | 手腕 | 俯仰 | 控制手腕上下俯仰 |
| S4 | 手腕 | 旋转 | 控制手腕旋转 |
| S5 | 夹爪 | 开合 | 控制夹爪张开/闭合 |

## PWM 与角度关系

- **S1 (大臂)**：中位1500时机械臂垂直向上。从1500向500或2500变化时，机械臂从前方向后旋转。
- **S0 (底盘)**：控制整体左右旋转，1500时居中。

## 盖章动作序列

盖章动作分三个阶段执行：

```
阶段1：中位 → 位置1
  - S1: 1500 → 920  (大臂前倾)
  - S2: 1500 → 1655 (小臂配合)
  - 持续时间: 800ms

阶段2：下压
  - S3: 1500 → 1630 (手腕下压)
  - 持续时间: 600ms

阶段3：抬起回到中位
  - 所有舵机 → 1500
  - 持续时间: 800ms
```

关键值：
- `STAMP_POS1 = {0:1500, 1:920, 2:1655, 3:1500, 4:1500, 5:1500}`
- `STAMP_WRIST_DOWN = 1630` (手腕下压目标)
- `STAMP_DOWN_PWM = 2400` (大臂下压力度)
- `STAMP_UP_PWM = 800` (大臂抬起)
- `STAMP_WRIST_DOWN_PWM = 1200` (手腕下压)
- `STAMP_WRIST_UP_PWM = 1700` (手腕抬起)

## 标定页面测试

在 `/calibration` 标定页面中，点击"测试盖章"按钮可执行完整盖章序列。
